项目能 build、1800+ 测试通过、CLI 实测跑通。卡住上线的从来不是 bug，是范围一直在扩、"完成"没有定义。v0.1 的目标是**收敛到一条能讲清楚、已经跑通的路径，发出去**。

## v0.1 做什么（这就是全部）

一条 golden path：

```
ic → 粘贴 DeepSeek / Qwen Key（或选 Mock）→ 说需求 → 看 diff → 1 写入 / 2 预览 / 3 取消
```

承诺给用户的能力，只有三件：

1. **改代码** — 理解项目，按需求精确修改，输出可审查的 diff。
2. **验证** — 改完自动跑检查并报告结果。
3. **记忆** — 记住项目与偏好，跨会话复用。

支持的模型服务：DeepSeek、Qwen、OpenAI、Claude、Mock。
