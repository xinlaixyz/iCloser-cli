# S20.3 底部固定操作面板

日期：2026-05-15
负责人：dev1
优先级：🔴 P0
估时：5h
依赖：S20.1

## 问题

用户必须记忆所有 `/` 命令名称。常用操作（写入、预览、撤销、提交）需要完整输入命令，效率低。文件列表、验证结果等上下文信息散落在对话历史中，翻找费时。

## 实现

### 文件: `src/cli/tui.ts` (扩展)

继承现有 `TuiScreen`，增加：

```typescript
interface BottomPanel {
  type: 'shortcuts' | 'files' | 'status' | 'verify';
  title: string;
  items: PanelItem[];
  actions: PanelAction[];
  height: number;          // 动态计算
}

interface PanelItem {
  key?: string;
  label: string;
  detail?: string;         // 灰色附加信息
  status?: 'ok' | 'fail' | 'running' | 'pending';
  checked?: boolean;
}

interface PanelAction {
  key: string;             // 快捷键字母
  label: string;
  action: string;          // 动作标识
}
```

**面板状态机:**

| 上下文 | 面板类型 | 触发条件 |
|--------|---------|---------|
| `shortcuts` | 命令快捷栏 | 空闲，无 pending files |
| `files` | 待写入文件 | state.pendingFiles.length > 0 |
| `status` | AI 执行状态 | streamState === 'loading' 或 'streaming' |
| `verify` | 验证结果 | verifyResult 存在 |

### 文件: `src/cli/repl.ts` (集成)

```typescript
function renderBottomPanel(): void {
  if (state.pendingFiles.length > 0) return renderFilePanel();
  if (streamState !== 'idle') return renderStatusPanel();
  if (lastVerifyResult) return renderVerifyPanel();
  return renderShortcutPanel();
}
```

**键盘绑定:**
- 输入框为空时：字母键直接触发对应 action
- 输入框有内容时：字母键正常输入
- `Ctrl+P`：打开命令面板

### 文件: `src/cli/theme.ts` (新增)

```typescript

```
