# S20.9 Tab 补全增强

日期：2026-05-15
负责人：dev1（全部独立）
优先级：🟢 P3
估时：1h

## 问题

当前补全只支持 slash 命令的前缀匹配，不支持文件路径、task ID 等上下文补全。

## 实现

增强 `replCompleter` (repl.ts)：

```typescript
function enhancedCompleter(line: string): [string[], string] {
  const tokens = line.split(/\s+/);
  const lastToken = tokens[tokens.length - 1] || '';

  // 1. 命令补全 (已有)
  if (line.startsWith('/')) return commandCompletions(line);

  // 2. 文件路径补全 (/write src/co<Tab> → src/config.ts)
  if (lastToken.includes('/') || (tokens[0] === '/write' || tokens[0] === '/diff')) {
    return filePathCompletions(lastToken);
  }

  // 3. Task ID 补全 (ic st <Tab> → 列出最近 task ID)
  if (['ic st', 'ic g', 'ic d', '/status'].some(c => line.startsWith(c))) {
    return taskIdCompletions(lastToken);
  }

  // 4. 上下文关键词补全
  return contextKeywordCompletions(lastToken);
}
```
