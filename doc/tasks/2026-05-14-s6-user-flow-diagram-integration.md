日期：2026-05-14

## 背景

用户提供了任务循环流程图：用户输入 Prompt → 收集上下文 → 执行操作 → 验证结果；验证完成则任务结束，需要继续则回到上下文收集，用户可随时干预。

## 交付

- 状态机增加用户干预：补充信息、调整方向、中断任务。
- 状态机增加验证分支：完成、继续循环、询问用户。
- 文档补充产品流程图逻辑。
- 测试覆盖用户干预路径。
