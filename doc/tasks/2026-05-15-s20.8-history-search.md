# S20.8 历史搜索增强

日期：2026-05-15
负责人：dev1（全部独立）
优先级：🟢 P3
估时：1h

## 问题

当前 ↑↓ 逐条翻历史效率低，无法搜索特定历史命令。

## 实现

### Ctrl+R 历史搜索

```
  ╭─ 历史搜索 ────────────────────────────────────────╮
  │ > config                                           │
  │ ─────────────────────────────────────────────────── │
  │ 12:30  帮我把 src/config.ts 改成连接池              │
  │ 12:05  ic config --list                            │
  │ 11:48  更新 docker-compose 配置                    │
  │ Ctrl+R 切换  │  Enter 填入  │  Esc 取消            │
  ╰────────────────────────────────────────────────────╯
```

- 从 `state.conversation` 中提取所有 user 消息作为历史
- 模糊匹配描述内容
- Enter → 填入输入框，Esc → 取消

### 验收

```
测试: Ctrl+R 打开搜索面板
测试: 输入过滤词 → 实时过滤历史
测试: Enter → 填入输入框 (不发送)
测试: 空历史 → 显示 "无历史记录"
```
