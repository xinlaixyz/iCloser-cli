```
  │ /scan           扫描项目并更新索引                  │
```

### 键盘绑定 (面板模式)
- 字符键 → 追加到 filter，实时过滤
- Backspace → 删除 filter 字符
- ↑↓ → 导航选项
- Enter → 执行选中命令
- Esc / Ctrl+P → 关闭面板

