## 五、执行流水线

```
任务输入
  → 对话式澄清（模糊任务 AI 先提问）
  → 任务记忆学习（从历史获取策略建议）
  → 影响感知上下文（AST 依赖图查找影响范围）
  → 文件感知规划（已知文件跳过探索）
  → 并行文件探索
  → 系统逐步执行 + 决策点 + 自动回退
  → Token 预算超限 → 提前合成
  → AI 合成代码变更
  → 自审查（完整性/正确性/一致性/安全性）
  → 编译验证 + 自动修复（tsc --noEmit）
  → 语义验证（import/reference 检查）
  → 快照备份
  → 依赖排序写入
  → 自动测试生成（缺失检测→AI生成→空检测→重生成）
  → 增量索引刷新
  → Verifier: compile + lint + test（3轮修复）
  → Gate: 6道门禁
  → 任务记忆记录（供未来学习）
  → 报告
```
