## 四、架构变更

### 控制反转：AI 驱动 → 系统驱动

```
之前: AI → 调用工具 → 系统执行 → AI 再调用 → 重复失败 → 超时

之后: 系统 → AI 生成计划 → 系统逐步执行 → 空结果自动回退
```
